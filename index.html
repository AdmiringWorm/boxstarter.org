Boxstarter

Repeatable, reboot resilient windows environment installations made easy using Chocolatey packages. When its time to repave either bare metal or virtualized instances, locally or on a remote machine, Boxstarter can automate both trivial and highly complex installations. Compatible with all Windows versions from Windows 7/2008 R2 forward.

Download v2.13.0

Why Boxstarter?

Checkout the features that Boxstarter provides on top of Chocolatey to ensure 100% uninterrupted Windows environment installs.

Run Remotely

Need to install a bunch of machines or repeatedly setup a VM? Invoke Boxstarter commands to install environments on remote machines. Supports all VM technologies and deep integration with Hyper-V and Windows Azure VMs.

Store Configs in a Gist

You can use Boxstarter to install a complete environment or install a small set of tools and windows settings with absolutely no software pre installed and configuration scripts stored in a gist.

Simplify Package Creation

Boxstarter composes an environment using Chocolatey style NuGet Packages. Boxstarter provides commands to make package authoring, testing and publishing a snap.
